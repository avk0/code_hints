## Python Environment

Create virtual environment \
`virtualenv -p python310 env` \
`python -m venv ./venv`


Activate vitrual env. \
`source env/bin/activate` - Linux \
`.\env\Scripts\Activate.ps1` - Windows

Deactivate vitrual env. \
`deactivate`
