# git
List of all remotes \
`git remote`

Change remote url \
`git remote set-url origin NewURLHere`

reset changes to last commit on remote (HEAD) \
`git fetch --all`
`git reset --hard origin/<branch_name>`
`git merge origin/main` - merge main commits to current branch
